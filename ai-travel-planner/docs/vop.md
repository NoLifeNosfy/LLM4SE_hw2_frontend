
# 🎧 前端 API 接口文档（语音识别模块）

## 📘 模块说明

本模块用于前端录制语音后，将音频文件上传给后端（FastAPI），
由后端调用百度语音识别 API 进行识别，并返回识别文本。

---

## 🔹 1. 上传录音并获取识别结果

### **接口地址**

```
POST /api/vop
```

### **接口说明**

上传用户录制的音频文件（推荐 `.wav` 格式，16kHz、16bit、单声道），
后端接收后直接将音频转发给百度语音识别 API，返回识别结果。

---

### **请求头**

| 参数名            | 值                     | 必填 | 说明       |
| -------------- | --------------------- | -- | -------- |
| `Content-Type` | `multipart/form-data` | ✅  | 上传文件表单类型 |

---

### **请求参数**

| 参数名    | 类型           | 必填 | 说明               |
| ------ | ------------ | -- | ---------------- |
| `file` | File（或 Blob） | ✅  | 录制的音频文件，格式推荐 WAV |

---


---



# 接口返回示例（前端接收到）

```json
{
  "err_no": 0,
  "err_msg": "success.",
  "result": ["请问我能帮您什么？"],
  "sn": "fae2108b-8dfc-4d4b-8bfb-3d3bdbbc91a5"
}
```



## 🔹 4. 注意事项

| 项目   | 说明                        |
| ---- | ------------------------- |
| 录音格式 | WAV（PCM 编码，16bit 单声道）     |
| 采样率  | 16000Hz（百度推荐）             |
| 文件大小 | 建议 ≤ 10MB                 |
| 上传接口 | 支持 HTTPS，必须带 Token（如后端鉴权） |
| 请求超时 | 建议设置 30 秒（长音频识别可能较慢）      |
